## student workflow
- open the repository directly on GitHub, not from local files  
- watch videos from the playlists, starting with statistical mathematics for machine learning  
- each student summarizes the video they finish  
- upload the summary as a markdown file or PDF on the repository  
- follow daily updates to track progress  
- collaboration is encouraged; students may create a private WhatsApp or Discord group if needed  
- aim to finish all videos within one month  
- the most important thing is to summarize and practice actively, using pen and paper  
- for any questions, contact the supervisor or team members

---

## how to contribute
- fork the repository  
- summarize concepts in markdown or PDF  
- add examples and visualizations if possible  
- submit pull requests  
- review and refine collectively
